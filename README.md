# hsorter

Приложение для ведения библиотеки фильмов и сериалов в Linux Mint с использованием SQLite и PyGObject (GTK 3).

## Запуск

```bash
python3 hsorter.py
```

## Возможности
- Три панели: библиотека тайтлов, карточка тайтла, медиафайлы.
- Фильтры по названию, тегам и статусам.
- Статусы качества в виде чеклистов.
- Автоматическое чтение характеристик видео через MediaInfo (CLI или `pymediainfo`).
- Перетаскивание файлов для изображений/видео через GTK drag-and-drop.

## Зависимости
- `python3-gi` и GTK 3 (PyGObject).
- `pymediainfo` и/или `mediainfo` для извлечения данных видео.
