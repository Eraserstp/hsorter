# hsorter

Приложение для ведения библиотеки фильмов и сериалов в Linux Mint с использованием SQLite.

## Запуск

```bash
python3 hsorter.py
```

## Возможности
- Три панели: библиотека тайтлов, карточка тайтла, медиафайлы.
- Фильтры по названию, тегам и статусам.
- Статусы качества в виде чеклистов.
- Автоматическое чтение характеристик видео через MediaInfo (CLI или `pymediainfo`).
- Перетаскивание файлов для изображений/видео при наличии `tkinterdnd2`.

## Зависимости (необязательно)
- `pymediainfo` и/или `mediainfo` для извлечения данных видео.
- `Pillow` для предпросмотра обложек.
- `tkinterdnd2` для drag-and-drop.
