# hsorter

Приложение для ведения библиотеки фильмов и сериалов в Linux с использованием SQLite и PyGObject (GTK 3).
Найвабкожено с использованием Codex и Deepseek - используйте на свой страх и риск.

## Запуск

```bash
python3 hsorter.py
```

## Импорт из Ant Movie Catalog (AMC)

```bash
python3 amc_import.py /path/to/export.xml /path/to/hsorter.sqlite
```

Параметр `--dubs` запишет дубликаты в файл `dublicates.csv`.

## Возможности
- Три панели: библиотека тайтлов, карточка тайтла, медиафайлы.
- Фильтры по названию, тегам и статусам.
- Сортировка по названию и дате добавления.
- Статусы качества в виде чеклистов.
- Автоматическое чтение характеристик видео через MediaInfo (CLI или `pymediainfo`).
- Перетаскивание файлов для изображений/видео через GTK drag-and-drop.
- Диалог подробностей видео: миниатюры, треки, комментарии и изображения видеофайла.

## Зависимости
- `python3-gi` и GTK 3 (PyGObject).
- `pymediainfo` и/или `mediainfo` для извлечения данных видео.
